## Functions

This is a nodejs web application to learn easily.

- **language : Nodejs**
- **front-end : jquery, bootstrap**
- **back-end : express**
- **view : hbs**
- **file upload : multer**
- **HTTP : XMLHttp, fetch, ajax, axios**
- **websocket : socket.io**
